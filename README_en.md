# Go Kit

[中文](./README.md) | English

Go Kit is a lightweight Go toolkit providing some common utility functions.

## Modules Overview

| Module | Description | Usage                                    |
| --- | --- |------------------------------------------|
| `arg` | Struct-based command-line argument parser | [Usage](./arg/README_en.md)              |
| `collectionx` | Convenient collection operation utilities | [Usage](./collectionx/README_en.md)         |
| `debug` | Easy-to-use debugging utilities like log throttling | [Usage](./debug/README_en.md)               |
| `errorx` | Error creation and wrapping utilities, simplifying error handling | [Usage](./errorx/README_en.md)              |
| `executor` | Simple concurrent task executor like loop tasks | [Usage](./executor/README_en.md)            |
| `framework/structure` | Utilities for simplifying struct parsing | [Usage](./framework/structure/README_en.md) |
| `params` | Parameter validation utilities | [Usage](./params/README_en.md)              |
| `perf` | Performance analysis utilities like execution time tracker | [Usage](./perf/README_en.md)                |
| `stringx` | Common string processing functions | [Usage](./stringx/README_en.md)             |

## Installation

```bash
go get github.com/l0neman/go-kit
```
