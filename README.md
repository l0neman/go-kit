# Go Kit

中文 | [English](./README_en.md)

Go Kit 是一个轻量级的 Go 工具集。提供了一些常用的工具函数。

## 模块概览

| 模块 | 描述 | 用法 |
| --- | --- | --- |
| `arg` | 通过结构体描述命令行参数的解析器 | [用法](./arg/README.md) |
| `collectionx` | 集合操作的便捷工具函数 | [用法](./collectionx/README.md) |
| `debug` | 简单易用的调试工具，如日志优化 | [用法](./debug/README.md) |
| `errorx` | 错误的创建和包装功能，简化错误处理 | [用法](./errorx/README.md) |
| `executor` | 简单的并发任务执行器，如循环任务 | [用法](./executor/README.md) |
| `framework/structure` | 简化结构体解析的工具 | [用法](./framework/structure/README.md) |
| `params` | 用于参数校验的工具 | [用法](./params/README.md) |
| `perf` | 性能分析工具，如代码执行时间跟踪器 | [用法](./perf/README.md) |
| `stringx` | 常用的字符串处理函数 | [用法](./stringx/README.md) |

## 安装

```bash
go get github.com/l0neman/go-kit
```
